<br>CSS also know as CAscading Style Sheet.
<br>it used to add the styling in our WEb Page.
<br>There is three way to connect CSS to Your HTML file.
<br>These are-
<br>
<br>
***Inline CSS*** applies styles directly to an HTML element using the style attribute.  This is best for unique, single-element styling but is not recommended for larger projects as it mixes presentation with content. 

<br>p style="color: blue; font-size: 16px;">This is a blue paragraph.</p>

<br>

***Internal CSS*** defines styles within a <style> element placed in the <head> section of the HTML document.  This method is useful for styling a single page with a unique design. 
<br><head>
<br><style>
<br>body { background-color: powderblue; }
<br>h1 { color: red; }
<br></style>
<br></head>

<br>

***External CSS*** links a separate .css file using a <link> tag in the <head> section.  This is the best practice for maintainability, reusability, and caching across multiple pages. 

<br><head>
 <br>link rel="stylesheet" type="text/css" href="styles.css">
<br></head>
